# Simple Games: Rock Paper & Scissors

This project showcases classic game, **Rock Paper & Scissors**

## Features

### Rock Paper Scissors
- **Player vs Computer**: Play against a randomly choosing value.
- **Score Tracker**: Keeps track of wins, losses, and ties.
